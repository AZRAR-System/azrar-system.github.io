📋 How to add it in Outlook

Outlook on the web (outlook.com / Microsoft 365):
  1. Sign in to the mailbox.
  2. Settings ⚙️ → Mail → Compose and reply → Email signature.
  3. Click “Copy signature” above, click inside the box, and paste (Ctrl+V).
  4. Tick “new messages” + “replies/forwards” → Save.
  5. Repeat for each shared mailbox (open it via your photo → Open another mailbox).
Outlook desktop (Windows):
  1. File → Options → Mail → Signatures…
  2. New → name it “AZRAR” → paste the signature → set as default → Save.
